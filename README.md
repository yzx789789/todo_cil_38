# todo_cli_38
C语言版简易命令行待办清单工具，适配 Dev-C++ 5.11，无额外依赖。

## 项目信息
- 项目名称：todo_cli_38
- 开发者：yzx789789
- 学号后两位：38
- 功能：添加/查看/标记完成/删除待办（支持自动扩容、输入校验）
- 编译环境：Dev-C++ 5.11（内置 MinGW 编译器）

## 功能特点
✅ 支持待办添加（非空校验，避免无效输入）  
✅ 支持待办查看（显示完成状态：[完成]/[未完成]）  
✅ 支持标记待办为完成（序号校验，避免崩溃）  
✅ 支持删除待办（自动整理后续待办顺序）  
✅ 动态扩容（初始容量5，满后自动翻倍）  
✅ 内存安全（动态内存分配后及时释放，无泄漏）  
✅ Dev-C++ 5.11 直接编译运行（无需额外配置）

## Dev-C++ 5.11 编译运行步骤
1. 打开 Dev-C++ 5.11，点击「文件 → 新建 → 源代码」；
2. 复制 `todo.c` 完整代码，粘贴到文件中，保存为 `todo.c`（需在 `todo_cli_38` 文件夹下）；
3. 配置编译标准：点击「工具 → 编译选项 → 编译器 → 勾选「支持 C99 标准」（兼容 C89）；
4. 编译：快捷键 `F9`（下方输出窗口显示「编译成功」即可）；
5. 运行：快捷键 `F10`，按命令行菜单提示操作。

## 操作说明
1. 运行后输入 1-5 选择功能，按回车确认；
2. 添加待办：输入 1 → 输入待办内容 → 回车；
3. 查看待办：输入 2 → 显示所有待办及序号；
4. 标记完成：输入 3 → 输入待办序号 → 回车；
5. 删除待办：输入 4 → 输入待办序号 → 回车；
6. 退出工具：输入 5 → 确认退出。
